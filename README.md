# portfolio
This is my personal website showcasing what do I do all day!

---

## Tech Stack
- **Framework**: React + TypeScript
- **Bundler**: Vite
- **Styling**: Vanilla CSS (Premium Dark Theme)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
